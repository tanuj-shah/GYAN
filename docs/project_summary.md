# GYAN Project Summary

1. **Project name.** GYAN (Global Youth Alliance for Nepal)
2. **One-line pitch.** A premium digital platform connecting global Nepali youth for national development through education, mentorship, and partnership.
3. **Primary goal.** Uniting Nepali youth worldwide to drive national development.
4. **Main user.** Nepali youth (students, professionals) and Admins.
5. **Top 3 features.** Global Member Directory, Event Management System, and Vision 2035 Proposal Engine.
6. **Critical flow.** User visits home -> Registers account -> Completes Profile -> Submits Vision 2035 Proposal.
7. **Tech stack.** PHP (Vanilla), MySQL, Tailwind CSS, Alpine.js.
8. **Auth/roles.** User, Admin (pending/active status).
9. **Big constraint.** Premium/Elite Design Aesthetics.
10. **Wanted output.** Generate features/code matching high-end design specs.
11. **Tone/voice.** Elite, Professional, Inspirational, Cinematic.
12. **Key Context.** Refer to `docs/project_documentation.md` for technical deep dives.
